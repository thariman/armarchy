Create scripts to install Omarchy on Archboot for ARM using Parallels Desktop (tested on MBP M4 Max).
Use the latest Archboot ARM ISO:
https://release.archboot.com/aarch64/latest/iso/

Prepare the environment using Archboot and the script from the PR below:
https://github.com/basecamp/omarchy/pull/1897

Modify the config.sh files as needed.
Then run ArchAuto.sh
